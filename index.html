Disclaimer

This DNS Speed Test tool runs entirely in your browser, sending live DNS-over-HTTPS (DoH) queries from your local environment. No server-side code is involved, and no personal data is collected.

Warm-Up Phase: Before timing each DNS server, the script primes connections and caches to minimize overhead from TCP/TLS handshakes.

Time Measurement & Browser Overhead: We use fetch and performance.now() to measure DoH round-trip times. Results may be higher than raw DNS tools due to HTTP/TLS overhead and browser behavior.

Timeouts & Unavailable Results: Each query times out after 5 seconds. If a server does not respond in that window, it appears as Unavailable.

Influence of Local Conditions: Network latency, bandwidth, and browser load can cause variability. Run multiple tests at different times for a broader picture.

Results are relative comparisons under your current conditions and should be used as guidance when choosing a DNS provider.
